** how program get converted?

=> In .NET, source code is first compiled into MSIL (Microsoft Intermediate Language). 
This MSIL is then executed by the CLR (Common Language Runtime). The CLR uses
a JIT (Just-In-Time) compiler to convert MSIL into machine code at runtime.
The JIT compiles only the required methods when they are called. The CLR
also provides services like memory management through Garbage Collection,
exception handling, and security.
